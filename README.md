# Snack Mini Store 🍫

Simple E-Commerce Snack Store Website.

## Features

* Product List
* Add to Cart
* Remove Items
* Checkout System
* Price in Philippine Peso ₱

## Technologies

* PHP
* MySQL
* HTML
* CSS
* Sessions

## Database

Import `snackstore.sql` into phpMyAdmin.

## Deployment

Upload to Render or ByetHost.

## Products

* Chips
* Chocolate
* Cookies
* Soft Drinks
